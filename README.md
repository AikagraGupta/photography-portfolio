# Photography Portfolio

A cinematic photography and videography portfolio built with Next.js and powered by a live Flickr feed.

## Features

- 3D hero gallery adapted from the imported v0 template
- Recent public photos pulled from Flickr at build time and revalidated hourly
- Editorial stills layout, archive grid, motion section, and contact CTA
- Remote Flickr image support configured for Next.js image optimization

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Checks

```bash
npm run lint
npm run build
```
